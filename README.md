# syncfolder
一个支持双向同步的笔记管理软件（目前仅支持 md 格式）

# build

## install dependency

```shell
sudo apt-get install cmake qt5-default qtcreator qtwebengine5-dev
```

```shell
git clone --recurse-submodules git@github.com:faywong/syncfolder.git
mkdir build
cd build
cmake ..
```
